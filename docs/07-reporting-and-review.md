# Lesson 7: Reporting & Review

**Goal:** Use built-in reports to review service, sales, and project activity, and wrap up the tutorial.

## Steps

1. From the main menu, open the **System** or **Reports** area to find the report list.

   ![Reports list](../assets/screenshots/07-reporting-and-review/01-reports-list.png)

2. Run a standard report such as a ticket summary or sales pipeline report.

3. Filter by date range, status, or resource to focus on what matters.

   ![Sample report](../assets/screenshots/07-reporting-and-review/02-sample-report.png)

4. Export or save the report view if you need to share results.

   > Remember to redact any client names, ticket details, or revenue figures before adding report screenshots to the repo.
