# Lesson 1: Getting Started

**Goal:** Understand what ConnectWise PSA is, who it's for, and how to access it before diving into the core features.

## What is ConnectWise PSA?

ConnectWise PSA (Professional Services Automation) is a business management platform built for Managed Service Providers (MSPs) and IT teams. It brings ticketing, sales, project management, time tracking, and billing into a single system so that service work can be tracked from first contact through to invoicing.

## What you'll need

- A ConnectWise PSA account (covered in the next lesson)
- A modern web browser
- Optional: a notepad for jotting down navigation paths as you explore
