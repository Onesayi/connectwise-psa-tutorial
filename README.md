# ConnectWise PSA — Beginner Tutorial

An unofficial, hands-on tutorial that walks through the core features of ConnectWise PSA for MSPs: account setup, ticketing, opportunities, and projects. Built as a learning project — all screenshots use demo or redacted data.

## What you'll learn

- Getting started and creating an account
- Navigating the dashboard
- Creating and managing service tickets
- Working with opportunities and sales
- Setting up and tracking projects
- Basic reporting

1. [Getting Started](docs/01-getting-started.md)
