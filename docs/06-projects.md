# Lesson 6: Projects

**Goal:** Set up a project, add phases and tickets, and track progress to completion.

## Steps

1. From the main menu, open the **Project** module and select **Project List**.

   ![Project list](../assets/screenshots/06-projects/01-project-list.png)

2. Click **New Project**, link it to a company, and choose a project template or board.

   ![New project](../assets/screenshots/06-projects/02-new-project.png)

3. Add phases and project tickets to break the work into manageable pieces.

4. Assign resources and track time against each ticket as the project progresses.

5. Update the project status and close it once all work is delivered.
