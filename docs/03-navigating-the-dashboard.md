# Lesson 3: Navigating the Dashboard

**Goal:** Get comfortable with the main ConnectWise PSA layout so you can find any module quickly.

## Steps

1. After logging in, take note of the top navigation bar and the main menu. This is how you reach every module.

   ![Dashboard overview](../assets/screenshots/03-navigating-the-dashboard/01-dashboard-overview.png)

2. Open the main menu and review the key areas: Service Desk, Sales, Project, Time & Expense, and Finance.

3. Locate the search bar and your user/notifications area in the corner.

4. Explore the "My Calendar" or "My Workflow" view to see what's assigned to you.

   ![Main menu](../assets/screenshots/03-navigating-the-dashboard/02-main-menu.png)
